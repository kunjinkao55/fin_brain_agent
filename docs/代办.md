## Phase 1.5：估值框架 + 回退鲁棒性（✅ 已完成，2026-07-23）

### 双锚点估值体系
- 主锚点：情景概率加权（机构标准多情景 DCF）
- 参考锚点：PE 乘数法（保守地板价）
- 结构突变检测：V 型反转时自动切换到年化 ROE + 最新季度数据
- 成长溢价扩展：B 级(≥5)→1.1, A 级(≥7)→1.3, S 级(≥9)→1.8

### 回退路径（3 轮 + ast 兜底）
1. 第 1 轮：完整 prompt + ANALYST_PROMPT → 原始 LLM → JSON 提取
2. 第 2 轮：精简 prompt（砍十步框架，只给数据+JSON 指令）
3. 第 3 轮：极简模板（字段模板 + 评分规则）
4. 兜底：`ast.literal_eval` 解析 Python 格式 dict
5. 空壳检测：AnalystOutput 全 null → 自动触发回退
6. 连接重试：`LLMFallbackChain` 对连接中断类错误 3 次退避重试

### 评分引擎时效修复
- 成长性：flash 提取容错（f_net==-100→回退季报对比）
- TTM EPS：跳过 flash 行 (`_快报源=True`)
- 结构突变：Q1 年化 ROE > 年报 ROE×2 → 自动覆盖
- 年化因子兜底：报告期字段未匹配时按日期月份推断
- 投资评级 ROE：同步使用结构突变修正后的 ROE

### LLM 输出格式稳定
- Few-shot 示例：`ANALYST_PROMPT` 新增完整单股 JSON 范例
- 双引号强制：4 处 prompt 显式标注 "双引号，禁止单引号"
- `[{...}]` 自动解包：接受数组包单对象格式
- Reporter 稀疏检测：有意义字段=0 → 视为解析失败

### 连接稳定性
- `LLMFallbackChain.invoke()` 对 `connection closed/reset/timeout/unexpected EOF` 类错误进行 3 次退避重试（3s/6s）
